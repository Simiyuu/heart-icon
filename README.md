# SVG Heart Icon

A live deployment of the Heart Icon project built for the freeCodeCamp curriculum. This project demonstrates how to render scalable vector graphics directly within an HTML document without relying on external image files.

**Live Demo:** [Insert your GitHub Pages link here once deployed]

## 🚀 Tech Stack
* HTML5
* SVG (Scalable Vector Graphics)

## 🧠 What I Learned
* Implementing the `<svg>` element to draw graphics natively in the browser.
* Defining a coordinate system and rendering scale using the `viewBox`, `width`, and `height` attributes.
* Tracing complex shapes and curves using the `<path>` element and the `d` attribute data.
* Applying inline styling attributes like `fill` directly to vector components.

## 🛠️ Setup
To view this project locally:
1. Clone the repository: `git clone https://github.com/Simiyuu/heart-icon.git`
2. Open `index.html` in your web browser.
